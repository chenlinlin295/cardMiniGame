## 4. 动物卡牌素材（头像层 + 底牌色层）

> 棋盘按关卡切换：普通 **12 列（4×3，2层）**；困难 **20 列（4×5，5~6层）**。每张牌最终呈现：圆角方 256×256，3D 厚边，柔和阴影。  
> **核心改动**：不再“动物=固定底色”，改为“动物头像层 + 底牌颜色层”组合。  
> 关卡不要求 36 种全上：建议 **普通关随机抽 8 种**、**困难关随机抽 18 种**。

### 4.1 分层资产与运行时组合规则

- 分层资产：
  1. 动物头像层：`animal_<name>_head.png`（36 种）
  2. 底牌颜色层：`card_base_<color_id>.png`（颜色池）
  3. 公共边框/高光层（可选）：`card_frame_common.png`
- 组合规则（必须执行）：
  1. 同一局内：不同动物不能使用同一底色  
  2. 同一局内：同一动物在所有牌上必须同一底色  
  3. 跨局：同一动物可更换底色，不做全局绑定
- 视觉原则：底色只用于“区分当局动物”，不改变动物本体配色，不做“猫变蓝毛、虎变白毛”这类错误表达。

### 10.8 牌量 / 难度联动参数（给关卡与出图）

> 以下参数用于保证“普通关轻量 + 困难关高压”与可消除规则一致，建议和 `GAME_DESIGN.md` 保持同步。

| 项目 | 规则 |
|------|------|
| 普通关牌量 | 固定 `N = 12 * 2 = 24` |
| 困难关总牌公式 | `N = 100 + x`（`x` 为 6 层列数） |
| 可全消约束 | `N` 必须是 3 的倍数 |
| 困难关合法总牌数 | 102 / 105 / 108 / 111 / 114 / 117 / 120 |
| 动物配比约束 | 每种动物数量必须为 3 的倍数 |
| 正式动物池 | 36 种（收集册进度 X/36；排行榜含“抓到几个/抓到几种”双页签） |
| 单局抽样 | 普通关抽 8 种，困难关抽 18 种 |
| 颜色规则 | 动物与底牌色解耦；同局内不同动物不同色、同动物同色；跨局可重映射 |
| 槽位规则 | 下方 7 槽单卡；每次入槽/回槽自动重排并让同动物相邻；满3张消除 |
| 复活规则 | 失败类型独立计次：超时可复活1次，槽满可复活1次 |
| 移出区显示规则 | 默认隐藏；移出技能或槽满复活后显示；不能遮挡棋盘可点牌 |
| 普通关建议 | 24 张，120 秒，复活加时 +30 秒 |
| 困难关建议 | 117 张，480 秒，复活加时 +90 秒 |

**单局抽样配比（可直接给关卡生成器）：**

- 普通 24（抽8种）：`3,3,3,3,3,3,3,3`
- 困难 117（抽18种）：`9,9,9,9,9,9,6,6,6,6,6,6,6,6,6,3,3,3`
- 移出技能：每次按 7 槽重排后顺序自动上移前端最多 3 张（FIFO，不手动选牌；不足3张则全上移）
- 槽满复活：将 7 槽卡牌全部上移到移出区，按 `3-3-1` 分摞（例如 6 张 = 下层3张+上层3张）
- 回放：移出区卡牌可点回 7 槽，仅可逐层点击每摞最上层
- 胜利：棋盘、7 槽、移出区三处卡牌全部清空

**给 AI 的高度提示词（整页/场景图可复用）：**

```text
normal board 4x3 columns with 2 cards each; hard board 4x5 columns with 5-6 cards each; top card clickable style
```

## 12. 验收清单

### 透明

- [ ] PNG 真实 alpha，非白底非棋盘格
- [ ] 边缘干净

### 对局

- [ ] 卡牌采用分层：`animal_*_head` + `card_base_*` 可正确组合
- [ ] 同局分色满足“不同动物不同色、同动物同色”

- [ ] 7 单卡槽 + 3 移出堆叠位
- [ ] 移出区默认隐藏，激活显示且不遮挡棋盘

- [ ] 胜利判定包含棋盘、卡槽、移出区全清空

- [ ] 设置/规则 96×96
- [ ] 中间 GIF 512×512 可循环
