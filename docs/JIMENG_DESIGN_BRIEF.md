## 1. 全局规范

### 1.1 平台与尺寸

| 项 | 值 |
|----|-----|
| 平台 | **仅微信小程序移动端** |
| 设计稿 | **750 × 1334 px**，@2x，约 9:16 |
| 逻辑基准 | 375 × 667 → 导出 750 × 1334 |
| 倍率 | **只要 @2x**，不要 @3x / 4K |

**禁止**：手机壳、系统状态栏、浏览器框、水印、二维码。

### 1.2 UI 原则：Icon 优先

| 位置 | 规则 |
|------|------|
| 首页顶栏 | 左 **设置 Icon**，右 **规则 Icon**，不要大文字按钮 |
| 对局顶栏 | 左 **暂停 Icon**，中 **倒计时文字**，右侧留空 |
| 对局底栏 | **移出 / 凑齐 / 洗牌** 仅 **Icon**，不要技能文字 |
| 首页底栏 | 排行榜 / 开始游戏 / 收集册（可 Icon+叠字，开始游戏最大） |

中文若 AI 易变形：**留空白区域由程序叠字**。

### 1.3 分关卡棋盘规格

- 对局页按关卡切换棋盘：
  - **普通关**：4 行 × 3 列 = 12 列；每列 2 张
  - **困难关**：4 行 × 5 列 = 20 列；每列 5~6 张
- 每列仅 **最顶一张** 可点击；列有 3D 厚度感。
- 关卡数值建议：普通 **24 张 / 120 秒**，困难 **117 张 / 480 秒**。

### 1.4 对局页垂直分区（自上而下，给 AI 标注用）

| 分区 | 约占比 | 内容 |
|------|--------|------|
| A 顶栏 | 8% | 暂停 Icon 左，倒计时中，右侧留空 |
| B 棋盘 | 42% | 普通 4×3（12列）/ 困难 4×5（20列）动物牌 |
| C 移出区 | 10% | 3 个堆叠位（默认隐藏，激活后显示，多层堆叠） |
| D 卡槽区 | 12% | 7 个透明玻璃单卡槽（每槽1张） |
| E 技能栏 | 14% | 移出 / 凑齐 / 洗牌 三 Icon |
| 余量 | 14% | 安全区、阴影 |

### 1.5 动物头像与底牌颜色（解耦规则）

- 动物头像与底牌颜色是两层资产：
  - 头像层：36 种动物
  - 底牌层：独立颜色池（多色）
- 对局按“激活动物集合”动态映射颜色：
  - 普通关激活 8 种动物
  - 困难关激活 18 种动物
- 映射约束：
  1) 同一局中，不同动物不能使用同一底色  
  2) 同一局中，同一动物必须保持同一底色  
  3) 跨局可重新分配颜色，不做全局绑定
- 牌型：圆角正方形、3D 厚橙边、中央 Q 版动物头像、柔和投影。

### 1.6 卡槽区与移出区

- **卡槽（7 个）**：真实半透明玻璃凹槽，单槽单卡，不做同类叠放角标。
- **重排规则**：每次入槽或回槽后，7 槽自动整理，相同动物相邻显示。
- **移出区（3 个堆叠位）**：在 7 卡槽 **上方**，默认隐藏；使用移出技能或槽满复活后出现。
- **不遮挡要求**：移出区不可遮挡棋盘可点击牌区域。
- **分摞规则**：槽满复活时把 7 槽卡牌全上移，按层分摞（如 3-3-1）。
- **回放规则**：移出区卡牌可点回 7 卡槽，但只能逐层点击每摞最上层。

### 1.7 收集册未获得 / 已获得

| 状态 | 视觉 |
|------|------|
| 未获得 | **纯黑卡片** + 中央 **白色大号问号 ?** |
| 已获得 | 彩色 3D 动物卡，高亮点亮（展示色可独立于对局分色） |

### 1.8 复活按钮统一外观

- **看视频复活** 与 **分享微信复活** 使用 **同一套按钮样式**（尺寸、圆角、厚度、颜色）。
- 差异仅：左侧小 Icon（播放三角 vs 分享箭头/气泡）+ 程序叠字。
- **放弃**：小号灰木牌按钮，与复活按钮明显区分。

### 1.9 转场弹层：3 秒自动

- 文案：**「大批量动物来袭！」**
- 中央大号 **3 → 2 → 1** 倒计时。
- **无**「继续挑战」按钮，**无**「困难关开启」副标题，**无**返回/关闭入口。
- 倒计时结束后强制进入困难关（用户只能进入第二关，设计稿可画「1」或进度条暗示）。

### 1.10 复活成功

- **不做**复活成功弹层/动画。
- 超时复活 = 加时；槽满复活 = 7 槽全上移（3-3-1 分摞），直接回到对局。

## 13. 设计稿验收清单

- [ ] 750×1334 @2x，微信小程序，无手机壳/水印
- [ ] 首页：左上设置 Icon、右上规则 Icon、中间动物动图、底栏三按钮
- [ ] 对局：左上暂停 Icon、中间倒计时、右侧留空；普通为 **12列（4×3）每列2张**，困难为 **20列（4×5）每列5~6张**
- [ ] 对局：3 个移出堆叠位在 7 卡槽上方
- [ ] 对局：7 个单卡槽，真实玻璃槽，每槽仅1张
- [ ] 移出区默认隐藏，仅在移出技能或槽满复活后出现，且不遮挡棋盘
- [ ] 7槽每次入槽/回槽后自动重排，同动物相邻；移出技能按重排后前端最多上移3张（FIFO）
- [ ] 胜利判定：棋盘、卡槽、移出区三处都清空
- [ ] 对局：底栏三技能 **仅 Icon**
- [ ] 动物牌来自 **36 种总池**，单局随机抽样（普通 8 / 困难 18）
- [ ] 同局分色满足：不同动物不同色、同动物同色
- [ ] 转场：「大批量动物来袭」+ **3 秒倒计时**，无继续按钮
- [ ] 复活：两复活按钮 **样式完全一致**
- [ ] 无复活成功弹层设计
- [ ] 收集册：未获得 = **黑底白问号**
- [ ] 无分数、无关卡号
- [ ] 排行榜含双页签：抓到几个 / 抓到几种（右侧值随页签切换）
